DNS используется не только для веб-сайтов, и существует несколько типов DNS записей. Мы рассмотрим наиболее распространённые из них, с которыми вы, скорее всего, столкнётесь.

**Запись A**

Эти записи преобразуются в IPv4-адреса, например 104.26.10.229

**Запись AAAA**

Эти записи преобразуются в IPv6-адреса, например 2606:4700:20::681a:be5

**Запись CNAME**

Эти записи указывают на другое доменное имя. Например, в интернет-магазине TryHackMe есть поддомен [store.tryhackme.com](http://store.tryhackme.com/), который возвращает запись CNAME [shops.shopify.com](http://shops.shopify.com/). Затем отправляется ещё один DNS запрос на [shops.shopify.com](http://shops.shopify.com/), чтобы определить IP-адрес.

**Запись MX**

Эти записи содержат адреса серверов, которые обрабатывают электронную почту для запрашиваемого вами домена. Например, ответ MX-записи для [tryhackme.com](http://tryhackme.com/) будет выглядеть примерно так: [alt1.aspmx.l.google.com](http://alt1.aspmx.l.google.com/). Эти записи также содержат флаг приоритета. Он указывает клиенту, в каком порядке следует обращаться к серверам. Это удобно, если основной сервер не работает и электронную почту нужно отправить на резервный сервер.

**Текстовая запись**

Записи TXT — это свободные текстовые поля, в которых можно хранить любые текстовые данные. Записи TXT можно использовать по-разному, но чаще всего их применяют для составления списка серверов, имеющих право отправлять электронные письма от имени домена (это может помочь в борьбе со спамом и поддельными электронными письмами). Их также можно использовать для подтверждения права собственности на доменное имя при регистрации в сторонних сервисах.
