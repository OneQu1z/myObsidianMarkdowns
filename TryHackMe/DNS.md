DNS (система доменных имён, Domain name system) позволяет нам легко взаимодействовать с устройствами в интернете, не запоминая сложные номера. Подобно тому, как у каждого дома есть уникальный адрес для отправки почты, у каждого компьютера в интернете есть свой уникальный адрес для связи с ним, который называется IP-адресом. IP-адрес выглядит следующим образом: 104.26.10.229. Это 4 набора цифр от 0 до 255, разделённых точкой. Когда вы хотите зайти на сайт, не очень удобно запоминать этот сложный набор цифр, и здесь на помощь приходит DNS. Поэтому вместо того, чтобы запоминать 104.26.10.229, вы можете запомнить [tryhackme.com](http://tryhackme.com/).
