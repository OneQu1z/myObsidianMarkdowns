**V**irtual **P**rivate **N**etwork (или **VPN** для краткости) — это технология, которая позволяет устройствам в разных сетях безопасно взаимодействовать друг с другом, создавая выделенный канал связи через Интернет (так называемый туннель). Устройства, подключённые к этому туннелю, образуют собственную частную сеть.
Например, напрямую взаимодействовать могут только устройства в одной сети (например, в корпоративной). Однако VPN позволяет соединить два офиса. Рассмотрим схему ниже, на которой изображены три сети:

Устройства, подключённые к сети № 3, по-прежнему являются частью сетей № 1 и № 2, но вместе они образуют частную сеть (сеть № 3), по которой могут обмениваться данными только устройства, подключённые через VPN.

Давайте рассмотрим некоторые другие преимущества VPN в таблице ниже:

| **Выгода**                                                               | **Описание**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Позволяет объединять сети, расположенные в разных географических точках. | Например, компаниям с несколькими офисами будет полезно использовать VPN, поскольку это позволит получить доступ к таким ресурсам, как серверы и инфраструктура, из другого офиса.                                                                                                                                                                                                                                                                                                                                                 |
| Обеспечивает конфиденциальность.                                         | VPN технология использует шифрование для защиты данных. Это означает, что данные могут быть расшифрованы только теми устройствами, с которых они были отправлены и на которые они должны быть доставлены, то есть данные не подвержены риску перехвата.<br><br>Такое шифрование полезно в местах с общедоступным Wi-Fi, где сеть не обеспечивает шифрование. Вы можете использовать VPN, чтобы защитить свой трафик от посторонних глаз.                                                                                           |
| Предлагает анонимность.                                                  | Журналисты и активисты используют VPN, чтобы безопасно освещать глобальные проблемы в странах, где свобода слова ограничена.<br><br>Обычно ваш трафик может просматриваться вашим интернет-провайдером и другими посредниками и, следовательно, отслеживаться.<br><br>Уровень анонимности, который обеспечивает VPN, зависит от того, насколько другие устройства в сети соблюдают конфиденциальность. Например, VPN, который регистрирует все ваши данные/историю, по сути, ничем не отличается от обычного VPN в этом отношении. |

VPN технологии совершенствовались на протяжении многих лет. Давайте рассмотрим некоторые существующие VPN технологии:

| **Технология VPN** | **Описание**                                                                                                                                                                                                                                                                                                                                   |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| PPP                | Эта технология используется в протоколе PPTP (см. ниже) для аутентификации и шифрования данных. VPN работают с использованием закрытого ключа и открытого сертификата (аналогично **SSH**). Для подключения закрытый ключ и сертификат должны совпадать.<br><br>Эта технология не способна самостоятельно покинуть сеть (не маршрутизируется). |
| PPTP               | **P**oint-to-**P**oint **T**unneling **P**rotocol (**PPTP**) — это технология, которая позволяет данным из PPP передаваться и выходить за пределы сети.<br><br>PPTP очень прост в настройке и поддерживается большинством устройств. Однако по сравнению с альтернативами он обеспечивает слабое шифрование.                                   |
| IPSec              | Internet Protocol Security (IPsec) шифрует данные с помощью существующего **I**nternet **P**rotocol (**IP**) фреймворка.<br><br>IPSec сложнее настроить по сравнению с альтернативами, но в случае успеха он обеспечивает надежное шифрование и поддерживается на многих устройствах.                                                          |
