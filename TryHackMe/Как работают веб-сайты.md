Веб-сайты в основном создаются с помощью:

- HTML — для создания веб-сайтов и определения их структуры
- CSS — для придания веб-сайтам красивого вида с помощью стилей
- JavaScript — для реализации сложных функций на страницах с использованием интерактивности

**H**yper**T**ext **M**arkup **L**anguage (HTML) — это язык, на котором пишутся веб-сайты. Элементы (также известные как теги) — это строительные блоки HTML-страниц, которые сообщают браузеру, как отображать контент. В приведенном ниже фрагменте кода показан простой HTML-документ, структура которого одинакова для всех веб-сайтов.
![](https://assets.tryhackme.com/additional/how-websites-work/example_html.png)

Структура HTML (как показано на скриншоте) состоит из следующих компонентов:

- Тег `<!DOCTYPE html>` определяет, что страница является документом HTML5. Это помогает стандартизировать работу в разных браузерах и указывает браузеру использовать HTML5 для интерпретации страницы.
- Тег `<html>` является корневым элементом HTML-страницы — все остальные элементы располагаются после него.
- Тег `<head>` содержит информацию о странице (например, заголовок страницы).
- Тег `<body>` определяет тело HTML-документа; в браузере отображается только содержимое внутри элемента body.
- Элемент `<h1>` определяет крупный заголовок
- Элемент `<p>` определяет абзац
- Существует множество других элементов (тегов), используемых для различных целей. Например, есть теги для кнопок (`<button>`), изображений (`<img>`), списков и многого другого.  
    

Теги могут содержать такие атрибуты, как атрибут class, который можно использовать для стилизации элемента (например, для изменения цвета тега) `<p class="bold-text">`, или атрибут _src_ , который используется для изображений и указывает их расположение: `<img src="img/cat.jpg">.`Элемент может иметь несколько атрибутов, каждый из которых предназначен для своей цели, например, <p attribute1="value1" attribute2="value2">.

Элементы также могут иметь атрибут id (`<p id="example">`), который уникален для каждого элемента. В отличие от атрибута class, в котором несколько элементов могут иметь один и тот же класс, у элемента должен быть уникальный идентификатор, чтобы его можно было идентифицировать. Идентификаторы элементов используются для стилизации и идентификации с помощью JavaScript.

Вы можете просмотреть HTML-код любого веб-сайта, щёлкнув правой кнопкой мыши и выбрав «Просмотреть исходный код страницы» (Chrome) / «Показать исходный код страницы» (Safari).
