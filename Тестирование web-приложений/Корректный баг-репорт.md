**Пример корректного баг-репорта**

|                          |                                                                                                                                                                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Заголовок**            | В поле Email отсутствует ограничение на максимальное количество символов в локальной части адреса электронной почты                                                                                                            |
| **Описание**             | В поле Email отсутствует ограничение на максимальное количество символов в локальной части адреса электронной почты. Можно использовать email с локальной частью, содержащей более 64 символов                                 |
| **Шаги воспроизведения** | 1. Открыть страницу [Ссылка на страницу].  <br>2. В поле Email ввести testtesttesttesttesttesttesttesttesttesttesttesttesttesttesttesttest@test.ru.  <br>3. Заполнить остальные обязательные поля.  <br>4. Нажать «Отправить». |
| **Актуальный результат** | Форма принимает адрес электронной почты с локальной частью, содержащей более 64 символов.  <br>_или_   <br>Регистрация пройдена успешно.                                                                                       |
| **Ожидаемый результат**  | Сообщение об ошибке с просьбой указать правильный/корректный email  <br>_или_   <br>Сообщение об ошибке «Некорректный email!»                                                                                                  |
